## Installation

```bash
pip install -r requirements.txt
```

## Usage

Firstly we need to start the [bridge](https://github.com/OwOHamper/IoT-B-S-H-device-controller/tree/main/Bridge) to start the server.
Afterwards start `main.py`:
```python
python main.py
```
